# 📈 金融即時監控儀表板 (Finance Dashboard)

這是一個基於 Python 與 Streamlit 開發的專業金融儀表板，提供即時匯率、大宗商品、全球指數及個股追蹤功能。

## ✨ 功能特色
- **即時匯率**：監控台幣對美金、歐元、日幣（高精度顯示）等主要貨幣。
- **能源與貴金屬**：追蹤黃金、白銀、布萊特原油及德州原油 (WTI)。
- **全球市場掃描**：分區監控亞洲、美洲及歐洲各大主要股市指數。
- **智慧個股監控**：
    - 支援中文名稱搜尋與自動代碼轉換。
    - 具備自動縮放刻度的智慧走勢圖。
    - **圖表標記**：自動標註期間內最高點與最低點。
- **績效追蹤**：所有標的皆提供「今日漲跌」與「年初至今 (YTD)」漲跌幅。
- **自動更新**：內建每分鐘自動刷新機制。

## 🚀 快速開始

1. **安裝依賴套件**：
   ```bash
   pip install -r requirements.txt
   ```

2. **執行應用程式**：
   ```bash
   streamlit run finance_app.py
   ```
